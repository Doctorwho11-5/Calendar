 Welcome to page 2!



In this page, we will learn the basics of how to start and where to start your coding
go to your desktop and right click.--- Click "new" and go to "text  document".
When you get to notepad, type at the top <!DOCTYPE html>. That's the beginning of your page. Now press 
enter and type </html>The <> brackets indicate a line of code and when you type the </> it ends 
the line of code. The computer needs to know when the line of code starts and ends so it doesn't get it 
confused with Text, like this, and other lines of code.
Your page should look like this so far. Note that you should always end "tags" with </tag>

<!DOCTYPE html>

</html>





Now go to the first line and press enter to go in between the two lines of code.
It should now look like this

<!DOCTYPE html>



</html>



Now in the middle of those two lines, type <head> </head>, press enter. In between the two "head" tags, type <title> </title>, 
you can type your title in between the two "title" tags.
Now under the "head" and "title" tags, type <body>,press enter, this is going to be where your text starts. Now type <p>, press enter"
</p>, this begins a paragraph.You can type words in between the <p> </p> tags, close off the body of the 
page by typing </body>under the </p> tag. 
Your page should now look like this.


<!DOCTYPE html>

<head>

<title>
</title>

</head>

<body>

<p>
Text Here
</p>

</body>

</html>
